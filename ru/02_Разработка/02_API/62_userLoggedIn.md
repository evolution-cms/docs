### Определяет, авторизован ли пользователь как менеджер или веб-пользователь. Возвращает информацию о нем

string userLoggedIn();

***

#### Формат данных результата:

	Array (
		[loggedIn] => 1
		[id] => 1
		[username] => siteadmin
		[usertype] => web
	)

***

#### Пример:

	$info = $modx->userLoggedIn();
	//вернет информацию о пользователе
